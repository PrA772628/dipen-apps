Excel and CSV Sheet Data Import

This module helps to Import Data From Excel and CSV Sheet to Create Custom Field in Model and Add data in model.

Features @Excel or CSV Sheet Data Import

  • Create new records in only allowed model by Adminstrator
  • Update old record using selection of first till fifth column as record selector option in record view
  • Can create new fields which having 'x_' as prefix
  • Fully Flaxible and having User Access rights
  • Can have option to only Create or Create/Update both
  • New Field creation can support 'many2one', 'integer', 'float', 'char', 'text', 'date', 'datetime', 'boolean' types of fields
  • Update the sheet and use worksheets name to import data
  • Supported format are .csv, .xls, .xlsx.
