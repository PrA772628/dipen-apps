Cancel Sale Order Pickings(1 steps,2 steps or 3 Steps)

This module allows to cancel 1 steps, 2 steps or 3 steps done pickings from sale order it selfs.

Features @Cancel Sale Order Pickings(1 steps,2 steps or 3 Steps delivery)

  • One stop to cancle whole sale order (Sale order, Deliveries, Invoices, etc..)
  • Allows to cancel 1 steps, 2 steps or 3 steps done pickings from sale order it self.
  • Create return picking if is there any "Done" oprations.
  • Also cancle the open or draft invoices.
