This module helps to Export Data From Odoo To Google Sheet For Write Model Data In Sheet.

Features @Google Sheet Data Export

  • Export/Update Any Model Data From Odoo To Google Spreadsheet.
  • Create new Sheet and worksheet using odoo it self and also get URL directly in odoo record.
  • You can Create worksheet or update it also.
  • Select your data using Filter option
  • There are selection of Models and Fields with sequence handler.
